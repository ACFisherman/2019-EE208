活力校园

隔着屏幕流口水！在交大，每天早餐不重样！

2019年06月20日 责任编辑：祁洁 

在交大，每天叫醒广大学子的

除了对未来的憧憬和梦想

还有交大不重样的特色早餐！

丰富多样的品类打动你的味蕾

咸甜麻辣的鲜香满足你的期待

在交大的每一天

都是被美食滋润的日子

#40天早餐打卡

为大家准备了一份精美的交大早餐攻略

请尽情享用~

文末重磅活动预告

网红牛油火锅蛋糕、钟水饺、成都汤圆

免费品尝

不可错过

玉兰苑 の 早点

交大“校园CBD”玉兰苑

是风味小吃云集的地方

地理便捷 环境优美

因此一直都是交大学子青睐的用餐地点

“巴比馒头”是早餐界的宠儿

虾仁包、荠菜包、小龙虾包

豆浆、玉米汁、黑米粥

丰富的品类，你会pick哪一款呢

PS：早晨7:40左右为人流高峰

可以适当错峰排队哦~

一餐 の 早点

俗话说 “近水楼台先得月”

一餐距离西区教学楼和宿舍楼较近

每到饭点人流就很大

土豆丝饼、大馄饨、南瓜粥

玉米粥、肉夹馍……

这里有诸多中式传统的早点

是否符合你的口味呢？

二餐 の 早点

二餐临近东区宿舍楼和教学楼

在人流量上也是不遑多让

煎饼果子和手抓饼的摊位

每一次都会排起长队

受欢迎的程度可见一斑呢

牛杂抄手、红汤小面

番茄刀削、甜咸豆花

……

品类齐全，吃完满满能量

别说了，小编在发愁明天早上吃啥了

三餐 の 早点

三餐临近捭阖塘

现在正值夏天

吃过饭后还可以欣赏含苞待放的荷花

小馄饨、小蛋糕、杂酱面

清粥、腐乳、咸菜

用简单健康的一份早餐

迎接新一天的开始吧~

四餐/五餐/六餐 の 早点

早晨去四餐、五餐、六餐可不是很近呢

但是这里的美味一定会让你不虚此行

口味正宗的热干面

皮薄汁多的汤包

豆浆、油条、烧麦

还有江浙沪一带特色的粢饭

一整天的阴霾都被扫去了

PS：五餐的东湖面馆

从这学期开始也供应早餐了哦~

哈乐 の 早点

哈乐餐厅坐落于西区宿舍群中

可谓是西区学子的后花园

这里的馄饨扬名在外

据说有许多同学其实并不住在西区

也曾早起特地去哈乐吃馄饨呢

甜魔、教超、罗森、全家

也许有的早上，你会比较匆忙

洗漱、穿搭、化妆

没有时间安安静静的坐下来享用一顿早餐

那么去便利店买早餐也是不错的选择哦

口感绵密的奶黄包

或是鳗鱼饭卷

还有各种口味的蛋糕面包

即使紧张忙碌也不要亏待自己

DIY，自己做早餐！

交大学子，心灵手巧，多才多艺

时间规划得宜时

也可以为自己定制一份早餐

自制的早餐口感营养一律满分

也不用费时费力去排队啦

用自己的厨艺开始一天的阳光明媚吧~

PS：自己做早餐一定要注意卫生和安全哦~
