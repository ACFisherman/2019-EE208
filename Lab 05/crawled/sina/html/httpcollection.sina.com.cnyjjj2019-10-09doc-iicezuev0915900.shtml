周春芽《桃花浅深处》迎秋

2019年10月09日 10:17 新浪收藏
微博 微信 空间 分享 添加喜爱

　　来源：苏富比(微博)亚洲

　　原标题：苏富比秋拍 | 当代艺术： 旋风一览晚拍巨匠精选杰作

　　香港苏富比当代艺术拍卖系列即将开锣，本次晚拍上呈拍品可谓艺坛群星荟萃，绝顶佳作万众期待。预展前夕，为解一众藏家苦等之奈，让我们带您速览今秋登场之精选拍品。

　　花卉蔬果，送夏迎秋

　　周春芽《桃花浅深处》

　　2010年作，油画画布，279.8 x 198 公分

　　估价：7，000，000 - 9，000，000港币

　　夏末秋至，周春芽《桃花浅深处》将以中国当代艺术焦点拍品登场晚间拍卖。引元稹《桃花》‘桃花浅深处，似勻深浅妆。春风助肠断，吹落白衣裳’，此刻会桃花，眨眼叹迎秋，《桃花浅深处》叮嘱观者光阴似箭，须珍重之。

　　刘炜《西瓜》与您别过炎温；草间弥生《南瓜》秋凉送暖。

　　刘炜《西瓜》

　　2004年作，200 x 60 公分

　　估价：3，000，000 - 5，000，000港币

　　草间弥生《南瓜》

　　1991年作，压克力画布，72.7 x 91 公分

　　估价：9，800，000 - 16，800，000港币

　　爱，无处不在

　　文人自古爱抒情，若诗人以字炼爱，艺术家笔下何尝不是一封封情书，一枚枚爱的烙印？约翰?柯林与格奥尔格?巴塞利茲均以爱妻为创作缪斯；而伊丽莎白·佩顿的《1999年5月（柯丝蒂和罗伊在西大街15号拥吻）》呈现了恋人间的细腻亲密。

　　伊丽莎白·佩顿（Elizabeth Peyton）

　　《1999年5月（柯丝蒂和罗伊在西大街15号拥吻）》

　　油画画板，42 x 35.8 公分

　　估价：3，200，000 - 4，800，000港币

　　约翰·柯林（John Currin）《忏悔者》

　　2004年作，油画亚麻布，107 x 86.6 公分

　　估价：8，000，000 - 12，000，000港币

　　格奥尔格?巴塞利兹（Georg Baselitz）《埃尔克，1965》

　　1996年作，400 x 300 公分

　　估价：4，000，000 - 6，000，000港币

　　西方巨匠，炼金炼珍

　　埃德·鲁沙（Ed Ruscha）《Point Blank》

　　1988年作，压克力油画画布

　　183.8 x 183.3 公分估

　　价：25，000，000 - 45，000，000港币

　　这次领衔晚间拍卖的西方代表乃当代传奇埃德·鲁沙，藏家即将见证这位蜚声国际的艺术家作品首登亚洲拍场。埃德?鲁沙生于1937年，对艺术创作热情从未间断，至今依然孜孜不倦，笔下作品媒材多样，并以其享负盛名的《城市之光》系列扬名国际，晚拍所推之《Point Blank》则为此系列的标志性作品。

　　《城市之光》系列取名自查理?卓别林（Charlie Chaplin）1931年执导的一出浪漫默片，系列作品几乎以单色成画，构图简约，与较早期的作品大相径庭。为了营造朦胧灯光点缀画面，鲁沙采用喷枪作画，并不断断探索这种技巧，创作硕果累累。《Point Blank》作于1988年，鲁沙于同时段以火红色字体创作之《城市之光》系列仅约十逾幅，本作为其一，突显罕有之余，亦展示了艺术家作品中典型的戏剧性张力和视觉震撼力，引人驻足凝视。

　　本次晚拍所推一众西方当代艺术家堪称当代艺术的炼金术士，他们承袭了中世纪炼金术的化学哲学思想，在艺术创作运用的媒材上尽展前瞻性创意。例如杰夫·昆斯人手吹制的玻璃凝视球、傅丹以金箔缀饰在曼谷市街上拾获的红牛饮料纸皮箱，以及亚历山大·考尔德以铁片铁丝巧制动态雕塑《蓝色上的黑、白、红》等，多元多样的艺术媒介在艺术家的冶炼下，促成了一件件艺珍的诞生。

　　杰夫·昆斯（Jeff Koons）《凝视球（梵谷的麦田群鸦）》

　　2015年作，油画画布，玻璃及铝材，116.8 x 224.2 x 37.5 公分

　　估价：7，000，000 - 10，000，000港币

　　亚历山大·考尔德（Alexander Calder）《蓝色上的黑、白、红》

　　1956年作，铁片、铁丝及颜料，42.9 x 42.5 x 15.2 公分

　　估价：3，500，000 - 5，000，000港币

　　人物具象，强势回归

　　当代市场再现人物具象热潮，从日本艺坛宠儿奈良美智及中国当代大师刘野，至比利时着名艺术家吕克·图伊曼斯，笔下作品从人出发，卡通形象或写实描绘，一一探讨人与人、人与事及人与物之微妙关系。

　　刘野《我是一个士兵》

　　1999年作，压克力画布，60 x 40 公分

　　估价：4，000，000-6，000，000港币

　　奈良美智《背后藏刀》

　　2000年作，压克力画布，234 x 208 公分

　　估价待询

　　吕克·图伊曼斯（Luc Tuymans）《你好！III》

　　2012年作，油画画布，126.9 x 175.1 公分

　　估价：5，600，000 - 7，600，000港币

　　东方挥毫，升形之上

　　东西两极相辉映，西方炼金，东方炼丹。具体派要员白发一雄和‘单色画’代表朴栖甫，至中国当代新晋艺术家黄宇兴的作品一同探索身体与思想的无极限。

　　黄宇兴《斗兽场》

　　2011年作，压克力油画画布，208 x 336 公分

　　估价：800，000 - 1，500，000港币

　　白发一雄《月氏》

　　1991年作，油画画布，160 x 130 公分

　　估价：6，000，000 - 8，000，000港币

　　朴栖甫《描法 No。 90-85》

　　1985年作，铅笔及油画画布，162 x 130 公分

　　估价：3，500，000 - 5，500，000港币

　　承接春季，特别企划

　　承接春季晚拍贝纳·维尼进行的即场艺术创作空前成功，本次晚拍将推二十世纪最具影响力的设计及建筑兼工程师让·普鲁维的《6 x 6 組合屋》，再现第二次时间大战后为舒缓法国紧急房屋短缺问题而创建出可高效拆卸的组合屋。

　　让·普鲁维（Jean Prouvé）《6 x 6 组合屋》

　　1944年作，金属、木材与玻璃，314.5 x 600 x 600 公分

　　估价：4，000，000 - 8，000，000 港币
