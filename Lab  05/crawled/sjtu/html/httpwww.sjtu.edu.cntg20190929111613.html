通知公告

关于2019年下半年校医院专家门诊时间安排的通知 ( 2019年09月29日 )

根据学校与附属医院的合作共建协定，校医院开设了专家门诊。现将2019年下半年校医院专家门诊时间地点及注意事项通知如下：

一、 就诊时间安排：

闵行校医院专家门诊时间安排在每周四(按单双周)的下午 13：30-15：45

徐汇校医院专家门诊时间：上午9:00-11:20,；下午13:30-15:30

二、 就诊地点安排：

闵行校医院专家就诊地点位于闵行校医院北楼二楼专家门诊部　

徐汇校医院专家就诊地点位于徐汇北二楼210室　

三、就诊注意事项：

1. 专家门诊向全校师生员工开放。 职工就诊时，请带好个人社保卡及就医记录册。学生就诊时，请带好校园卡及门诊病史。

2. 就诊时，请尽量带好个人病史资料、体检报告等与就诊疾病相关的材料，以供医生分析后提出诊疗意见。

3. 对于需要进一步检查与诊断的问题，可向接诊专家提出预约，在适当的时间去第六人民医院进行相关检查。

4. 学生如需去六院就诊，除向医生预约外，并请开具转院凭证。

5. 请各部门体检管理人员打印专家门诊时间，通过邮件通知体检后随访的员工。

6. 为更好地服务师生员工，欢迎大家为我们自己的医疗保健提出建设性的意见。联系信箱huaweijuan@sjtu.edu.cn

上海交通大学校医院

2019年9月27日
